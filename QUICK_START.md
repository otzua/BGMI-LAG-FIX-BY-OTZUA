**OLD WAY (10+ steps):**
Settings → Apps → Manage Apps → Search "joyose" → Select JoyOSE → Force Stop → Confirm → Clear Data → Confirm → Go to home screen → Find BGMI → Launch

**NEW WAY (4 steps):**
Open BGMI Booster → Open JoyOSE → Force Stop + Clear Data → Launch BGMI

**Time Saved:** ~45 seconds per session

→ Check if BGMI is installed properly

**App crashes on open?**
