# BGMI Booster (100% VIBE CODED BTW)

A professional, minimalist performance optimization utility for HyperOS and MIUI devices. This tool simplifies the process of optimizing the JoyOSE system component to unlock higher frame rates (up to 90 FPS) in Battlegrounds Mobile India.

## Features

- **True Black UI**: Optimized for OLED and AMOLED displays to save power and provide high contrast.
- **Minimalist Design**: A clean, professional interface using a custom Cream and Orange color palette.
- **Quick Shortcuts**: Direct access to JoyOSE system settings for faster clearing of data and force-stopping.
- **Auto-Detect**: Intelligent detection of BGMI and system components.

## How to Use

1. **Configure JoyOSE**: Tap the 'Configure JoyOSE' button. This will take you to the system settings for the JoyOSE app.
2. **Clear Data**: In the settings page, select 'Force Stop' and then 'Clear Data' (or 'Clear all data').
3. **Optimize**: Return to the BGMI Booster app.
4. **Launch**: Tap 'Launch BGMI' to start the game with the optimized system profile.
